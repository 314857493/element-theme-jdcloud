# element-theme-jdcloud

> 京东智联云定制主题

## Installation

执行下面命令

```shell
npm i element-theme-jdcloud
```

或者

```shell
yarn add element-theme-jdcloud
```

## Usage

Use Sass import

```css
@import "element-theme-jdcloud";
```

Or Use webpack

```javascript
import "element-theme-jdcloud";
```

## Import on demand

```javascript
import "element-theme-jdcloud/lib/input.css";
import "element-theme-jdcloud/lib/select.css";

// ...
```
